Applet question

Create an Applet showing the string "Hello World" which is embedded in the HTML page


import java.applet.Applet;
import java.awt.Graphics;

public class HelloWorldApplet extends Applet {
    public void paint(Graphics g) {
        g.drawString("Hello World", 20, 20);
    }
}

Output: Displays "Hello World" on the applet.

Create an Applet in Java to draw a polygon

import java.awt.Graphics;

        int[] xPoints = {50, 100, 150, 200, 250};
        int[] yPoints = {50, 100, 50, 100, 50};
        g.drawPolygon(xPoints, yPoints, 5);  // Draw polygon with 5 points
    }
}

Output: Draws a polygon on the applet.

Create an Applet in Java to draw 4 concentric circles


import java.applet.Applet;
import java.awt.Graphics;

public class ConcentricCirclesApplet extends Applet {
    public void paint(Graphics g) {
        for (int i = 0; i < 4; i++) {
            g.drawOval(50 + i * 20, 50 + i * 20, 100 - i * 40, 100 - i * 40);
        }
    }
}

Output: Draws 4 concentric circles on the applet.
